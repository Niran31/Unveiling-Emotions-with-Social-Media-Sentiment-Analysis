Unveiling the Tweetverse: Sentiment Dashboard

This project is a web-based sentiment analysis dashboard that works entirely in the browser.  
It analyzes single sentences or a CSV file of tweets, classifying them into Positive, Negative, or Neutral sentiments.

Features
- Single Text Analysis → Type a sentence and get instant sentiment
- CSV Upload → Upload a file of tweets for bulk sentiment analysis
- Interactive Dashboard → View results in a table and doughnut chart
- No API key required → 100% client-side using JavaScript, Chart.js, and TailwindCSS
- Includes project report PDF with methodology and system design
